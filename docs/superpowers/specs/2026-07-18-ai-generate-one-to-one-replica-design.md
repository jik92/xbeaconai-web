# AI 创作 1:1 复刻设计

目标：复刻 `#/aigc-tools/generate?ws=0` 到本地 `/tools/ai-generate`，以 1440 × 900、DPR=2 为基准，还原会话栏、顶部过滤、输入卡片、参考素材、模型参数弹层、提交确认和 Mock 任务结果。参考图 `55959f3c-c275-45af-b5a9-9406461e91e6.png` 是 2056 × 652 的输入卡片局部裁图；只有输入卡片参与 1:1 坐标验收，会话栏、过滤器、弹层和结果区是遵循同一视觉语言的产品补充区域。

约束：

- 仅使用浏览器 Mock 数据。
- 不修改既有 API、OpenAPI、生成 SDK 或服务端任务接口。
- 不调用真实图片、视频、音频模型，不触发 Seedance 真实任务。

参考交互：

- 页面主体：左侧 120 宽会话栏，包含 `+ 新对话`、`默认创作`。
- 顶部右侧过滤条：搜索、类型、日期范围、收藏。
- 输入卡片：900 宽，包含参考素材按钮、富文本输入区域、展开按钮、底部参数条、提交按钮。
- 输入卡片所有尺寸均为 CSS px：默认约 900 × 204，宽高误差不超过 8 px，圆角 16 px；参考按钮位于左上、提示文案位于上部、展开按钮位于右上，参数条底部对齐，黑色圆形提交按钮位于右下，手动确认复选框在卡片外部右下方。控件顺序不得变化。
- 1440 × 900 坐标表：内容区左侧会话栏 120 px；输入卡片在剩余内容区水平居中，top=124 px、padding=16 px；参考按钮 56 × 70 px；提示区 left=82 px、top=16 px；底部参数条距卡片左右/底部均 16 px，控件高 36 px，提交按钮 34 × 34 px。卡片背景 `#fff`、页面主背景 `#f1f5ff`、边框 `#f1f1f1`、主要文字 `#252d39`、占位文字 `#c4c8cf`。占位文案固定为“使用 @快速调用参考内容，例如：@图片1模仿 @视频1的动作，音色参考 @音频1”。
- 默认视频生成：`字节Seedance 2.0`、`全能参考`、`9:16`、`720P`、`5s`、`1个`。`提交前手动确认` 默认不勾选，与参考图一致；不勾选时直接创建 Mock 任务，勾选时先弹确认框。
- 模型弹层：展示 Seedance 2.0、Seedance 2.0 Mini、Seedance 2.0 Fast，以及 Mock 占位模型说明。
- 图片生成保留 Seedream Mock 生成路径，参数支持比例、1K/2K、数量、随机种子。
- 提交：浏览器内异步 Mock 任务。完成结果以卡片展示；“查看结果”打开本地预览弹层；继续追问回填输入框，创建变体新增完成态卡片，收藏作品切换本地状态，下载生成本地 JSON Blob。
- Mock 状态固定为 `idle → confirming（仅勾选时）→ generating → completed`，使用本地定时器在约 900ms 内完成；刷新页面不保留任务。下载生成本地 JSON Blob，继续追问回填输入框，创建变体新增完成态卡片，收藏切换本地状态。

验收：

- 1440 × 900、DPR=2 下，输入卡片宽高、控件顺序、圆角和主要间距与参考的 CSS 像素换算值误差不超过 8 px。
- 关键弹层和提交确认可点击走通。
- E2E 必须覆盖：生成类型切换、Seedance 模型选择、比例/清晰度弹层、数量/时长弹层，以及勾选手动确认后的提交确认弹层。
- E2E 在页面完成既有登录态恢复后监听完整创作交互；若出现 `/api/models`、`/api/uploads`、`/api/*/jobs`、`/openapi.json` 或任何模型/任务接口请求则失败。应用壳已有的认证与通知请求、静态页面和 Vite 资源请求允许。
