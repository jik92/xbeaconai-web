# 基本使用方式

> 最后更新：2026-07-21

注册或登录后进入工作区。账号、创作点、资产、项目和任务均按用户隔离；充值订单为本地 Mock。

1. 从侧栏选择已开放的创作流程或工具。
2. 填写输入并选择已上传资产；需要视频模型时选择已验证的 Seedance 模型。
3. 提交后查看队列和 Worker 的真实任务状态。
4. 成功或部分成功后预览、下载或继续编辑；失败任务按提示重试，取消任务等待上游核对完成。

爆款二创、一键成片、口播脚本拥有独立项目流程；一键成片和口播脚本不能通过通用任务接口替代提交。素材、商品、音色和人像在资产库管理；私有对象以短期签名地址读取。

任务和产物会标记 `real`、`local`、`mock` 或 `mixed`。该标记是判断是否使用真实外部能力的依据，不能将 Mock 结果描述为真实模型输出。
