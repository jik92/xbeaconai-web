# 启用工具新建任务文案设计

## 范围

仅调整 `menuFeatures` 中当前为 `true` 的 AI 工具箱和实用工具页面：视频分割、视频混剪、音色克隆、字幕擦除、画质增强、视频提取。设置为 `false` 的页面不受影响。

## 文案规则

所有通用“新建任务”文案均由工具名称生成，格式为 `新建{工具名称}任务`。同一页面的顶部按钮、创建弹窗标题和空状态创建按钮必须使用相同文案。

使用现有模块配置中的 `label` 作为名称来源，避免另建工具名称映射。使用独立页面的工具以页面名称作为来源，不改变任务类型、提交参数或业务逻辑。

## 验证

- 组件测试覆盖动态文案。
- 运行 TypeScript 类型检查和生产构建。
- 默认不运行 E2E。
