# 爆款二创远端素材回源设计

## 问题

素材库支持浏览器直传私有 TOS。直传完成后数据库会保存素材记录，但 Worker 本机的 `.data/uploads` 不一定存在对应文件。爆款二创解析任务目前只读取本地路径，因此会在素材记录有效时仍报“视频素材文件不存在”；商品参考图也存在相同问题。

## 设计

- 保持 API 契约、素材 ID 和账号隔离校验不变。
- Worker 为每个解析任务创建独立临时目录。
- 视频和商品参考图均优先使用 `.data/uploads` 中安全且真实存在的本地文件。
- 本地文件不存在时，从私有 TOS 下载到任务临时目录后再交给 FFprobe、FFmpeg 和视频分析 Provider。
- TOS 未配置或下载失败时返回明确错误，不静默切换 Provider 或 Mock。
- 无论任务成功或失败，都清理任务临时目录。

## 验证

- 单元测试覆盖远端视频和商品图成功下载并参与解析。
- 单元测试覆盖本地缺失且 TOS 不可用的明确失败。
- 运行 TypeScript 类型检查和生产构建。
