# 一键成片左栏滚动修复设计

## 目标

修复 `/aigc/video-create` 新建项目左栏无法滚动的问题。顶部项目标题和底部生成按钮保持固定，仅中间参数区域滚动。

## 实现

- 页面网格和左栏显式建立可收缩的高度链：`height: 100%`、`min-height: 0`。
- 左栏裁剪自身溢出，避免展开参数面板后撑高页面网格。
- 参数容器使用纵向自动滚动，并保留横向裁剪及触控惯性滚动。
- 顶部标题和底部操作栏继续使用不可收缩布局，不进入滚动容器。
- 移动端单列布局继续使用页面级滚动，避免产生嵌套滚动冲突。

## 验证

- 在 `1440x900` 和 `1024x768` 下展开高级设置。
- 验证参数容器确实可滚动到底部。
- 验证滚动前后顶部标题和底部“生成脚本”按钮始终可见。
