# 一键成片批量生成分镜视频设计

## 目标

移除脚本区域顶部的整篇重新生成入口，在分镜素材列增加批量生成入口，通过一个紧凑弹窗配置视频参数并为所有待生成或失败分镜提交独立任务。

## 交互

- 脚本工具栏移除“重新生成”，保留复制脚本、逐段“换一版”和手动编辑。
- 分镜表格素材表头增加“批量生成”按钮；没有待生成或失败分镜时禁用。
- 弹窗使用 shadcn Dialog，列出仓库现有的三个 Seedance 模型。
- 支持 `9:16`、`16:9`、`1:1` 比例和 `480P`、`720P` 清晰度，默认沿用项目模型与比例，清晰度默认 `720P`。
- Seedance 原声固定关闭，后续使用独立 TTS 配音；时长自动读取每个分镜的实际时长并限制在 4～15 秒。
- 确认后关闭弹窗并显示各分镜独立生成状态，不覆盖已成功生成或手动替换的素材。

## 服务端

- 新增项目级批量生成接口，校验项目归属、模型、比例和清晰度。
- 服务端只筛选 `pending` 与 `failed` 分镜，并为每个分镜创建独立 BullMQ Job。
- 批量参数作为本次任务覆盖值写入 Job，不修改项目表单默认值。
- 返回已提交 Job 列表；没有符合条件的分镜时返回明确冲突错误。

## 验证

- API/Store 测试覆盖用户隔离、筛选规则和独立任务数量。
- 页面静态测试覆盖脚本按钮移除、批量入口和 shadcn Dialog。
- 重新生成 OpenAPI/SDK，运行相关单测、类型检查和生产构建；不运行 E2E。
